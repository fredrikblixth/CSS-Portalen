Transform

Kombinerar man transform och transition så kan man göra animationer. I följande exempel kombinerar jag transform och transition.

Kod:

#rotate:active, #rotate:hover { transition:all 2s; transform:rotate(5deg); }

Klicka på mig! Jag roterar!

Kod:

#scale:active, #scale:hover { transition:all 2s; tranform:scale(0.5); }

Klicka på mig! Jag blir mindre!

Kod:

#skew:active, #skew:hover { transition:all 2s; transform:skew(10deg); }

Klicka på mig! Jag vrids!
