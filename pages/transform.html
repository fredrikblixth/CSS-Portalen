Transform

1. Rotate

Rotera ett element. Ange antal grader.

Kod:

#rotate { transform:rotate(5deg); }

Jag är roterad!

2. Scale

Scale förminskar eller förstorar ett element.

Kod:

#scale { tranform:scale(0.5); }

Jag är förminskad!

3. Skew

Skew "vrider" elementet åt ett visst håll. Ange i grader.

Kod:

#skew { transform:skew(10deg); }

Jag är vriden!
